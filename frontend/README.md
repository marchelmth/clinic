# Klinik Sehat Frontend

Frontend Klinik Sehat menggunakan React dan Vite.

## Project Structure

```text
/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── services/
│   ├── styles/
│   ├── App.jsx
│   └── main.jsx
├── index.html
├── vite.config.js
└── package.json
```

## Commands

| Command | Action |
| :-- | :-- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server |
| `npm run build` | Build production files to `dist/` |
| `npm run preview` | Preview the production build |
